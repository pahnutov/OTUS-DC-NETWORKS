## IP Plan

Для DC1 выделяется подсеть <b> 10.0.0.0/12 </b>
Диапазон IP адресов <b> 10.10.0.1 - 10.15.255.254 </b>

### Правила распределения адресов

IP = 10.Dn.Sn.X/31, где:

|Параметр|Назаначение|Диапазон|
|---|---|---|
Dn|номера сервисов внутри ЦОДа|0-15|
Sn|номер spine|1-255|
X|значение по порядку|1-255

где Dn: <br>
0 – loopback1 <br>
1 – loopback2 <br>
2 – p2p links 

### Распределение адресов

|Имя хоста|Тип хоста|Loopback1 IP|
|---|---|---|
DC1S1|Spine|10.0.1.1/32
DC1S2|Spine|10.0.2.1/32
DC1L1|Leaf|10.0.x.x/32
DC1L2|Leaf|10.0.x.x/32
DC1L3|Leaf|10.0.x.x/32
